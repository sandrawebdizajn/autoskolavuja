# Auto škola Vuja — sajt

Statički sajt za Auto školu „Vuja" i „Vuja Duo", Niš.

## Sadržaj
- `index.html` — cela stranica (jedna strana sa sekcijama: O nama, Obuka, Galerija, Cenovnik, Kontakt)
- `images/` — logo i fotografije galerije

## Pokretanje
Otvorite `index.html` u pregledaču. Nema build koraka ni zavisnosti — jedini eksterni resurs su Google Fonts.

## Objava na GitHub Pages
1. Napravite repozitorijum i otpremite sadržaj ove fascikle u koren.
2. U `Settings → Pages` izaberite granu (npr. `main`) i folder `/root`.
3. Sajt će biti dostupan na `https://<korisnicko-ime>.github.io/<repo>/`.
